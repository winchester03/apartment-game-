Apartment Game v0.2

Upload the CONTENTS of this folder to the root of your GitHub repository.

Controls:
- Drag anywhere to look around.
- Tap a visible point on the floor to walk there.

Includes:
- Four-wall enclosed room
- Ceiling and floor
- Door and window
- Basic lighting
- Wall/furniture collision
- Placeholder couch
- Drag-to-look
- Tap-to-walk
